# My Item

## Описание (Description)

Это проект на базе Electron с AI-бэкендом и веб-фронтендом. Приложение предоставляет функциональность для работы с элементами и интеграцией ИИ.

This is an Electron-based project with an AI backend and web frontend. The application provides functionality for working with items and AI integration.

## Установка (Installation)

1. Клонируйте репозиторий: `git clone <repository-url>`
2. Установите зависимости для основного проекта: `npm install`
3. Установите зависимости для бэкенда: `cd backend && npm install`
4. Установите зависимости для фронтенда: `cd frontend && npm install`

1. Clone the repository: `git clone <repository-url>`
2. Install dependencies for the main project: `npm install`
3. Install dependencies for the backend: `cd backend && npm install`
4. Install dependencies for the frontend: `cd frontend && npm install`

## Запуск (Running)

1. Запустите бэкенд: `cd backend && npm start`
2. Запустите фронтенд: `cd frontend && npm run dev`
3. Запустите основное приложение: `npm start`

1. Start the backend: `cd backend && npm start`
2. Start the frontend: `cd frontend && npm run dev`
3. Start the main application: `npm start`

## Структура проекта (Project Structure)

- `backend/` - Серверная часть с AI функциональностью
- `frontend/` - Веб-интерфейс на React/Vite
- `main.js` - Основной процесс Electron
- `preload.js` - Прелоад скрипт для безопасного доступа к API

- `backend/` - Backend with AI functionality
- `frontend/` - React/Vite web interface
- `main.js` - Electron main process
- `preload.js` - Preload script for secure API access
